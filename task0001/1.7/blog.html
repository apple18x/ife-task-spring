input默认有padding和margin数值

这里是文章的副标题

ie6-9、opera-10.51：

padding:1px;margin:0;firefox3.6+:margin:0;padding:0;

webkit内核的浏览器： 


                    input[type="radio"], input[type="checkbox"] {
                        margin: 3px 0.5ex;
                        padding: initial;
                    }
                    input[type="button"], input[type="submit"], input[type="reset"] {
                        padding: 2px 6px 3px 6px;
                    }
                    input[type="range"] {
                        padding: initial;
                        margin: 2px;
                    }
